Получение списка валют¶

Для получения списка валют необходимо создать класс, имплементирующий BaseCurrenciesRequester, а так же класс CurrenciesDict: .. code-block:: python

requester = CBRCurrenciesRequester() c = CurrenciesDict(requester)

c.request_currencies() cs = c.get_currencies() usd = c.get_currencies_by_id(‘R01235’)

c.set_tracked_currencies(‘R01335’, ‘R01239’, ‘R01235’) tracked = c.get_tracked_currencies()
